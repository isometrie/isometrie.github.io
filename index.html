    Thesis blackbox test (R&D)

    Introduction

    Compliance with quality standards in the area of web & mobile apps requires comprehensive testing of these applications. The testing effort required for this (test coverage) cannot be covered economically by manual testing alone.

    To improve test coverage, automated surface tests are used. These automated tests (test cases) were systematized during specification, quotation and test execution.

    Introduction

    At the Institute for Machine Tools and Production Engineering of the Technical University of Braunschweig, a robot-based tape applicator is currently being developed, which will be able to produce components automatically in the future using thermoplastic, unidirectional fibre composites.

    In order to monitor the production of the tape laying process and the resulting component quality, an image-based process monitoring system was design and build.

    Features

  • Research & development
  • Sensor data acquisition
